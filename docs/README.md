# Documentation

This folder contains all project documentation, including:

- **Technical documentation**: API references, architecture diagrams, and design documents
- **Academic papers**: Research papers, reports, and citations related to data standards
- **User guides**: Tutorials, how-to guides, and best practices
- **Meeting notes**: Records of team meetings and decisions
- **Presentations**: Slides and materials for presentations

## Structure

Organize documents by type:
- `papers/` - Academic papers and research documents
- `reports/` - Project reports and deliverables
- `api/` - API documentation
- `guides/` - User and developer guides
- `presentations/` - Presentation materials
