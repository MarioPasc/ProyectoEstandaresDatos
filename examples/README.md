# Examples

This folder contains example code demonstrating how to use the DataStandards project.

## Intended Contents

- **Basic usage**: Simple examples for getting started
- **Advanced usage**: Complex use cases and advanced features
- **Integrations**: Examples of integrating with other tools and libraries
- **Use cases**: Real-world scenarios and applications
- **Sample code**: Code snippets for common tasks

## Structure

Examples should be:
- Self-contained and runnable
- Well-commented and explained
- Use sample data from `data/samples/`
- Demonstrate best practices
- Include expected output or results

## Types of Examples

- `basic_validation.py` - Basic data validation example
- `custom_standards.py` - Creating custom data standards
- `batch_processing.py` - Processing multiple files
- `api_integration.py` - Using the API
- `visualization.py` - Visualizing validation results

## Running Examples

```bash
# Run an example
python examples/basic_validation.py

# Examples should work from the project root
cd /path/to/ProyectoEstandaresDatos
python examples/basic_validation.py
```
